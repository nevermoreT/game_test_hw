# 游戏平台快速入门指南

## 🚀 快速开始

### 1. 安装要求
- Python 3.6+
- Pygame

### 2. 安装步骤
```bash
# 安装Pygame
pip install pygame

# 进入游戏平台目录
cd game_platform

# 启动游戏
python main.py
```

## 🎮 游戏列表

### 贪吃蛇
- **类型**: 动作游戏
- **操作**: 方向键控制移动
- **目标**: 吃食物，避免撞墙

### 卡牌Roguelike
- **类型**: 策略卡牌
- **操作**: 数字键1-9使用卡牌，E键结束回合
- **目标**: 击败所有敌人

## 📖 操作说明

### 游戏中心
- **鼠标左键**: 选择游戏
- **ESC**: 退出

### 贪吃蛇
- **↑↓←→**: 控制移动
- **ESC**: 返回主菜单
- **空格**: 重新开始

### 卡牌Roguelike
- **1-9**: 使用卡牌
- **E**: 结束回合
- **ESC**: 返回主菜单
- **空格**: 继续下一战

## 💡 游戏技巧

### 贪吃蛇
1. 不要贪心
2. 提前规划路线
3. 利用墙壁转向

### 卡牌Roguelike
1. 合理使用能量
2. 优先防御
3. 观察敌人意图
4. 积累护甲

## ❓ 常见问题

**Q: 游戏无法启动？**
A: 检查Python和Pygame是否正确安装

**Q: 显示乱码？**
A: 确保系统支持中文字体

**Q: 卡牌无法使用？**
A: 检查能量是否足够
